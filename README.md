# 🎬 Layar Tancap

a Simple Movies REST API Plugin

## 🔨 Installation

- Composer Update : `composer update`
- NPM Install : `npm i`
- Grunt Build : `npx grunt`
  - Refactor Dot Framework : `grunt shell:dot_refactor`

## 🚦 API Endpoint

- `GET` `/wp-json/rymera/v1/movies?`
  - `page=` : `1`
  - `offset=` : `0`
  - `posts_per_page=` : `-1`
  - `category=` : `0`
  - `orderby=` : `date`
  - `order=` : `DESC`
  - `include=` : `7,10,12`
  - `exclude=` : `7,10,12`
  - `meta_key=` : `year`
  - `meta_value=` : `2007`
  - `suppress_filters=` : `true`
